1.API、名称、描述
User.Login、登录接口、用于验证并登录用户

2.接口URL：
http://104.194.79.57/demo/?service=User.Login

3.请求方式：
POST&GET

4.参数说明：
参数名字    类型  是否必须    默认值    范围   说明
Email       字符串 必须                      用户邮箱
password    字符串 必须                      用户密码

5.返回值说明：
返回字段        类型   说明
msg             字符串 提示信息
code            整型   操作码，1表示登录成功，0表示登录失败
info            对象   用户信息对象
info.userID     整型   用户ID
info.nickname   字符串 用户昵称
info.Email      字符串 用户邮箱


6.示例：
注册账号
http://104.194.79.57/demo/?service=User.Login&Email=taotao@taotao.com&password=111111
JSON:
{
"ret": 200,
"data": {
    "msg": "登录成功！",
    "code": "1",
    "info": {
        "userID": "26",
        "nickname": "taotao",
        "Email": "taotao@taotao.com"
    }
},
"msg": ""
}
