1.API名称、描述
Post.PostReply
帖子的回复-单个帖子的回复操作

2.接口URL：
http://dev.wuanlife.com/?service=Post.PostReply

3.请求方式：
POST

4.参数说明：
参数名字    类型  是否必须    默认值    范围      说明
post_id     整型    必须                          帖子ID
text        字符串  必须                        回复内容
user_id     字符串  必须                        回复人ID

5.返回值说明：
返回字段            类型        说明
post_base_id        整型       帖子ID
user_base_id        整型       发帖人ID
replyid             整型       回复人ID
text                字符串     回复内容
floor               整型       回复楼层
createTime          日期       回帖时间

6.示例：
回复帖子
http://dev.wuanlife.com/?service=post.Postreply&post_id=25&text=666666&user_id=5
JSON:
{
    "ret": 200,
    "data": {
        "post_base_id": 25,
        "user_base_id": "5",
        "replyid": "5",
        "text": "666666",
        "floor": 4,
        "createTime": "2016-04-22 20:31:06"
    },
    "msg": ""
}
